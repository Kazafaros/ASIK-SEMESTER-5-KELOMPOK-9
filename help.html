1. Memulai Analisis

Untuk memulai analisis HSI, ikuti langkah-langkah berikut:

  1. Buka halaman Analisis
  2. Isi semua parameter lingkungan
  3. Atau gunakan data contoh untuk mengisi dengan data preset

2. Memahami Hasil

Setelah analisis selesai, Anda akan melihat:

  • Nilai 0-1 yang menunjukkan kualitas habitat (≥0.75 = High, 0.45-0.75 = Medium, <0.45 = Low)
  • Daya dukung ekosistem berdasarkan HSI Score
  • Grafik yang menunjukkan kontribusi setiap parameter terhadap HSI
  • Peta interaktif dengan heatmap HSI di area Selat Sunda

3. Perbandingan Skenario

Gunakan fitur Perbandingan untuk membandingkan hingga 3 skenario berbeda:

  • Buat 2-3 skenario dengan parameter berbeda
  • Klik Bandingkan untuk melihat perbandingan visual
  • Analisis grafik dan tabel perbandingan untuk mengambil keputusan

Parameter Lingkungan

Suhu Permukaan Laut (SST)

Rentang: 18-32 °C
Unit: Derajat Celsius (°C)
Deskripsi: Suhu permukaan laut mempengaruhi metabolisme organisme laut dan distribusi spesies. Suhu yang terlalu tinggi atau rendah dapat mengurangi kesesuaian habitat.

Klorofil-a

Rentang: 0.1-3 mg/m³
Unit: Milligram per meter kubik (mg/m³)
Optimal: ~1.2 mg/m³
Deskripsi: Indikator produktivitas primer dan ketersediaan makanan. Nilai optimal menunjukkan ekosistem yang produktif.

Salinitas

Rentang: 30-37 PSU
Unit: Practical Salinity Unit (PSU)
Optimal: ~34 PSU
Deskripsi: Konsentrasi garam terlarut dalam air laut. Salinitas mempengaruhi osmoregulasi organisme laut.

Kedalaman

Rentang: 10-150 m
Unit: Meter (m)
Optimal: ~60 m
Deskripsi: Kedalaman air mempengaruhi penetrasi cahaya, tekanan, dan distribusi organisme. Kedalaman optimal mendukung keanekaragaman hayati.

Fitur Biogeografi

Fitur Biogeografi membantu Anda memprediksi jenis ikan dan organisme laut yang sesuai dengan kondisi lingkungan tertentu berdasarkan data real dari 12 spesies ikan Indonesia dengan parameter biologis.

Cara Menggunakan

  1. Buka halaman Biogeografi
  2. Masukkan HSI Score (0-1) dan parameter lingkungan:
    • Suhu Permukaan Laut (°C)
    • Salinitas (ppt)
    • Kedalaman (m)
  3. Isi area lokasi
  4. Klik Prediksi Spesies Ikan untuk melihat hasil
  5. Lihat daftar ikan yang sesuai dengan score kesesuaian (0-100%)

Interpretasi Hasil

  • Score ≥ 80% Sangat sesuai - Kondisi optimal untuk spesies ini
  • Score 60-80% Cukup sesuai - Kondisi masih mendukung pertumbuhan
  • Score <60% Kurang sesuai - Kondisi marginal untuk spesies ini

Database Spesies Ikan

Biogeografi menggunakan database 12 spesies ikan Indonesia dengan parameter real:

  • Pelagis (HIGH HSI) Tongkol, Cakalang, Tembang, Layang, Kembung
  • Demersal (MEDIUM HSI) Tenggiri, Pari, Kerapu, Bawal Hitam
  • Bentik (LOW HSI) Udang Windu, Teripang, Kerang, Rajungan

FAQ

Apa itu HSI (Habitat Suitability Index)?

HSI adalah indeks yang mengukur kesesuaian habitat untuk organisme laut berdasarkan parameter lingkungan. Nilai HSI berkisar dari 0 (tidak sesuai) hingga 1 (sangat sesuai).

Bagaimana cara interpretasi HSI Score?

High (≥0.75) Habitat sangat sesuai, daya dukung tinggi
Medium (0.45-0.75) Habitat cukup sesuai, daya dukung sedang
Low (<0.45) Habitat kurang sesuai, daya dukung rendah

Dari mana data parameter diambil?

Data dapat diinput manual atau menggunakan data contoh. Untuk aplikasi nyata, data dapat diambil dari data spasial di Copernicus, NOAA, BMKG, atau pengukuran langsung di lapangan.

Apakah hasil prediksi akurat?

Prediksi HSI menggunakan model yang mempertimbangkan berbagai parameter. Akurasi bergantung pada kualitas data input. Model ini dirancang untuk memberikan estimasi awal dan harus divalidasi dengan data lapangan.

Bagaimana cara export hasil analisis?

Setelah analisis selesai, klik tombol Excel" di card hasil untuk mengekspor. File Excel akan otomatis terunduh dengan nama berdasarkan tanggal. File Excel berisi beberapa Ringkasan, Parameter Input, Kontribusi Parameter, dan Detail Perhitungan.

Apakah data saya tersimpan?

Ya, semua prediksi otomatis tersimpan di browser localStorage anda dan dapat dilihat di halaman Riwayat. Data hanya tersimpan lokal di browser Anda.

Bagaimana rumus perhitungan HSI Score?

HSI dihitung menggunakan Weighted Average Method dengan memberi bobot berbeda untuk setiap parameter:

• Suhu Permukaan Laut (SST): 30%
• Klorofil-a: 25%
• Salinitas: 25%
• Kedalaman: 20%

Setiap parameter dinormalisasi ke skala 0-1 berdasarkan nilai optimalnya, kemudian dikalikan dengan bobotnya, dan dijumlahkan untuk menghasilkan HSI Score final (0-1).

Rumus & Metode Perhitungan HSI

Metode Weighted Average

Platform ini menggunakan Weighted Average Method untuk menghitung HSI Score. Metode ini memberikan bobot berbeda untuk setiap parameter lingkungan berdasarkan pengaruhnya terhadap kesesuaian habitat.

Bobot Parameter

Suhu Permukaan Laut (SST) 30%
Klorofil-a (CHL) 25%
Salinitas 25%
Kedalaman 20%
Total 100%

Langkah-Langkah Perhitungan

  1. Normalisasi Parameter Setiap parameter dikonversi ke skala 0-1 berdasarkan nilai optimalnya
  2. Weighted Sum Setiap nilai normalisasi dikalikan dengan bobotnya kemudian dijumlahkan
  3. HSI Final Hasil penjumlahan dibagi total bobot menghasilkan HSI Score (0-1)

Langkah-Langkah Perhitungan Detail

Langkah 1: Normalisasi Parameter

Setiap parameter dikonversi ke skala 0-1. Untuk parameter dengan nilai optimal (Chlorophyll-a, Salinity, Depth), semakin dekat ke nilai optimal, semakin tinggi skornya. Untuk SST, digunakan range optimal 27-29°C yang memberikan nilai maksimal.

Langkah 2: Kalkulasi Weighted Sum

Setiap nilai normalisasi dikalikan dengan bobot parameter-nya, kemudian dijumlahkan:

WeightedSum = (SST × 0.30) + (CHL × 0.25) + (SAL × 0.25) + (DEP × 0.20)

Langkah 3: HSI Score Final

HSI = WeightedSum (karena total bobot = 1.0)

Contoh Perhitungan Lengkap

Berikut contoh perhitungan HSI dengan parameter yang optimal:

Data Input:
• SST: 28°C (optimal)
• CHL: 1.2 mg/m³ (optimal)
• Salinity: 34 PSU (optimal)
• Depth: 60 m (optimal)
Nilai Normalisasi:
SST normalized = 0.9 (dalam range optimal)
CHL normalized = 1.0 (tepat di optimal)
SAL normalized = 1.0 (tepat di optimal)
DEP normalized = 1.0 (tepat di optimal)
Hasil Perhitungan:
WeightedSum = (0.9 × 0.30) + (1.0 × 0.25) + (1.0 × 0.25) + (1.0 × 0.20)
WeightedSum = 0.27 + 0.25 + 0.25 + 0.20 = 0.97
HSI = 0.97 → HIGH (Habitat Sangat Sesuai)

Penjelasan Normalisasi Parameter

Setiap parameter memiliki metode normalisasi berbeda berdasarkan karakteristiknya:

SST (Suhu Permukaan Laut)

  • Range: 18-32°C
  • Optimal: 27-29°C (memberikan normalized ~0.9-1.0)
  • Metode: Semakin mendekati range optimal, nilai normalisasi semakin tinggi

Chlorophyll-a (Produktivitas Primer)

  • Range: 0.1-3.0 mg/m³
  • Optimal: 1.2 mg/m³ (normalized = 1.0)
  • Formula: normalized = max(0, 1 - |nilai - optimal| / optimal_range)

Salinity (Salinitas)

  • Range: 30-37 PSU
  • Optimal: 34 PSU (normalized = 1.0)
  • Formula: normalized = max(0, 1 - |nilai - optimal| / optimal_range)

Depth (Kedalaman)

  • Range: 10-150 m
  • Optimal: 60 m (normalized = 1.0)
  • Formula: normalized = max(0, 1 - |nilai - optimal| / optimal_range)

� Catatan Penting

  • Validasi Hasil prediksi HSI harus divalidasi dengan data lapangan untuk memastikan akurasi
  • Kontekstual Nilai optimal dapat berbeda untuk spesies atau ekosistem yang berbeda
  • Dinamis Parameter dan bobot dapat disesuaikan berdasarkan penelitian terbaru
  • Komprehensif HSI hanya salah satu indikator; analisis ekosistem memerlukan pendekatan multi-disiplin
